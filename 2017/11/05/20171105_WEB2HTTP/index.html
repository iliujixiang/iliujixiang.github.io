新生大学课程《Web与网络基础》第2课：《HTTP基础》

一、课前闲聊

徐老师已经多年不用windows了，现在主要是用 Mac 电脑。对于编程开发而言，什么是重要？什么最难？就是编程环境的配置。什么样的电脑配置编程环境最简单的？mac电脑。以至于现在产生了一个产业，就是在网页上直接编程。徐老师表示：一个不会配环境的人，他就很有可能学不会编程。

Mac 电脑的操作系统在设计理念上与windows有很大的不同。windows电脑需要经常杀进程，麦克电脑可以不用杀进程，可以不用关机。Mac 电脑可以很大的提高效率。

徐老师使用麦克电脑之后，还培养了一个很重要的习惯，就是付费的习惯。付费就是少时间啊。

二、由请求说开去

下面开始讲课

请求

现在很少有不用网络请求的应用了。现在的应用，连日历，图片，都跟网络有关。一个小小的请求，能给我们带来哪些思考？应该注意哪些东西？

在浏览器打上网址，回车，它会自动在前面加上http或者https。http是一个协议。

协议

什么是协议？协议是计议和商议，在法律上还是合同的意思。是指双方约定的条款。在计算机的世界里充斥着协议。

函数的三要素：一名字，二参数，三返回值。这就是定义函数者和使用函数者的一种协议。你只有遵从这个协议，才可能把这个函数调用起来。

写一个程序的目的是什么？写一个函数的目的是什么？函数的目的是计算，程序也是。展现出来的结果都是计算出来的。阿里云的口号，一切都是为了计算。我们写程序的目的就是为了计算。
浏览器和服务端进行交互的目的是什么？交换数据，交往就要遵循一定的规则。学习编程就是学习一套规则。

协议的组成

再以新生大学的网址（URL或者叫uri）为例，讲一下一个协议应该包含什么？

一是名称，第二个是目标或当事人，第三个是具体的事情。

比如http是名称，www.xinshengdaxue.com是目标，lesson1是具体事情。
如果没有后面的lesson1呢，它也是有具体的事情。在JS里面没有返回值，就是返回值。

还有没有其他的协议。有，比如mailto、ftp、news、ssh等。最重要的两种协议，http，ftp。https不是另一种协议，它是http的加强版。大家要把重点放在http上。

三、传输过程

你在浏览器上发出了一个请求，服务器给你了一个返回，那么问题来了，这个请求是从哪里发出来的？是从你的浏览器发出来的吗？最终的出口肯定是你的网卡，然后你的网卡连着网线。知道这个过程可以使我们在学技术的时候，知道重点关注在哪。

一个网址，它是针对人的，是写给我们看的。但是这个东西一定会被转化，它要转化成对机器能看懂的语言。它转化的目的是什么？是为了让下一个处理模块能识别你的信息。

在未来的学习道路上，你一定会需要看别人的api，看别人的服务能给你提供什么能力。

github右下角有一个api，要学会看这种东西。比如，REST API V3。点开，其中可以看到GET。
我们的URl会被转换格式，从浏览器到网络模块到网卡，再到互联网。是先有的网络模块才有的浏览器。

网络模块的核心就是TCP/IP，他解决了最重要的两件事情，一是它可以给你找目标，二是把数据给你送出去。所有的操作系统都有网络模块，就是它的网络库。

TCP/IP本身可以看成是一个程序，它是用c语言开发的一个程序。

浏览器 - 网络模块 - TCP/IP - 网卡 - 节点 - 服务器。

浏览器到网络模块到tcp ip到网卡，再通过网线，通过节点，再到达服务器。

ping一下百度的IP： PING WWW.BAIDU.COM

一个问题是：给人方便就给程序不方便。需要解析，解析是一个过程，过程就会耗费时间。谁来解析？dns解析。

网络模块向dns请求，dns将ip返回给网络模块。这是一个耗时的过程。找到一个好的dns解析商非常重要。这也带来了一个产业。假如某一个国家把dns服务器给控制住了，那我们就上不去网了。

一个http从产生到转化，到dns解析得出IP，到TCP/IP，再到webapp，它会产生一个结果，返回给你。错误码是一个非常重要的东西，有非常多，10x 20x 30x 40x 50x，等。错误码也是一种协议。

小白学编程网站上有一篇文章，HTTP错误码汇总。

四、HTTP

经常用到的几个方法：

  • GET，希望给我一个网页；
  • POST，把数据送到服务器；
  • DELETE，希望删除数据；
  • PUT，对即有的资源做一个差异的变化；
  • ……

百分之九十的程序员用前两个。其它的就基本不怎么使用了。为什么有这种现象？因为大家不够重视，没有去用好的意识。

REST API 非常重要，要重视，以后可深入。

五、结语

今天的课程是希望给大家一个方向，知道把关注点放在哪儿。

不希望新生大学的学员，只知道post，get。

对数据的控制是一项重要的能力，对网络通道减少一点压力，也是对社会的贡献。
